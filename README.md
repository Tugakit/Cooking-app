Application de recettes avec une base de données

Objectifs du projet :
•	Création d’une application Excel de recettes,
•	Un fonctionnement simple et fiable pour une utilisation récurrente
•	Que l’application soit complète, avec le plus d’utilités nécessaires mais sans etre très complexe à utiliser et comprendre
•	Créer une application multiplateforme

A propos du projet :
Ce projet consiste à créer un tableur Excel pour sauvegarder des recettes et ses ingrédients, les méthodes et bien plus. Le tout pour permettre un stockage de recettes rapide avec toutes les informations requises et des fonctionnalités intéressantes.

Fonctionnement du projet.

La (ou les feuilles) Excel seront faites basées sur les données déjà formatés en tableaux en base de donnée pour ensuite les afficher correctement.  Le tout sera fait grâce au macros sur Excel et donc du VBA. Il y aura une feuille, qui sera utilisée comme un formulaire pour entrer les nouvelles recettes et les informations qui vont avec dans ce style.
 De l’autre côté il y a une grande liste dynamique qui affichera tout ce qui a été créé et stocké. Toutes les parties nécessaires pour créer les recettes dans le formulaire et les présenter. Ce sera dans un état brut sans interface ou triage. (Exemple en dessous)
